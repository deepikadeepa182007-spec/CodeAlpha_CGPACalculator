# CodeAlpha CGPA Calculator (C++)

## Description
This project is a CGPA Calculator developed using C++ and Object-Oriented Programming concepts.
It allows users to select courses from an available list, enter grade points and credit hours,
and calculates the final CGPA.

## Features
- Displays available courses
- User selects enrolled courses
- Input validation for grades
- Calculates CGPA
- Displays PASS/FAIL status
- Overall grade calculation

## Concepts Used
- C++ Basics
- Object-Oriented Programming (Class & Objects)
- Arrays
- Functions
- Input/Output Formatting

## How to Run
1. Compile the program using a C++ compiler
2. Run the executable
3. Select courses and enter grade details as prompted
